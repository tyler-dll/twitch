Tools I created to make moderating on Twitch easier. No login needed.

## Tools (live)

### Lurk Command Generator

Build a random !lurk command for StreamElements or Nightbot.
Includes platform switching and auto variable conversion.

Page: https://tyler-dll.github.io/twitch/lurk/

### Variables Documentation

Search common variables, see examples, learn how they can be used.
Includes links to official docs.

Page: https://tyler-dll.github.io/twitch/docs/

python -m http.server 8000

Then open:
